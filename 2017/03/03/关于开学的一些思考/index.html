关于开学的一些思考

发表于 2017-03-03 |

关于开学的一些思考

买书

纠结了好久的一本书：Python科学计算(第2版)，还是决定不买了。不是说买来不会读，不会去学习，而是觉得毕竟是作为工具的语言，而不是作为职业的语言，还是没那么多精力去专研它，或者说，解决眼前的问题它并不是迫切需要的

虽然没有书也是可以学习的，但买就是下了决心，因为我太知道自己是多么的懒惰。关于将来的职业，还是有一丝幻想成为程序员：一个懂土木业务的程序员。当然我也很难给出懂业务的程序员的定义，不过至少算是爱好的体现吧。很多东西都是入门即坑，科普又是另外一种素养。

科研

在老师的指导下，了解了一下机电相似理论，知道惯容器这个device可以封装intertia，从惯容器与阻尼器的结合到从机械车辆设备到土木工程设备类比，也算是一次自主的摸索，简单的问题经过论证有效后，嘿，您猜怎么着？关于位移放大来发挥阻尼器性能的研究国内外早已进行研究了，虽然只是想写个专利，但还是有一种挫败的感觉，可是科研的美不就是在这里吗？创新永远不是那么简单，但却要你有更多的智慧。

上课

新学期开始了，各种学科的选修课蜂拥而至，又回到了本科什么课都上的状态，各种课程都觉得对自己有用，但是现在的研究领域并不涉及。课程是冲突的，也有学分和时间的限制，所以也很难去平衡。但旁听确实是美好的体验，就如同刚上大学时希望读遍图书馆的书一样，最后一本也没读完。我太知道这种忙碌是怎样的一种后果，术业要有专攻，但也可能一条路走到黑。学生和学者还是不同的。

关于毕业

隐约觉得两年毕业还是有希望的，曾经一度放弃过这个想法，疯狂的想去调节课程安排，抽出时间去实习，而现实是各种冲突，而且想法多于行动。现在也算静下来一些，一切按两年毕业来准备，按部就班的进行每一项工作。要努力，不能心急。同时也要有信心！
