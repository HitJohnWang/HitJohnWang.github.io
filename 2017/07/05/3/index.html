愿我们都成为信息时代的独立阅读者

发表于 2017-07-05 |

热爱阅读从来不是我的习惯。

渴望阅读一直都是我的longfor。

不够热爱，是说自己阅读的永远不够多，渴望阅读，是说自己永远不会抓紧时间。

在很功利的时代，阅读是一件很难的事。
凡事只找解决的办法，让生活变得更加匆忙。
独立的阅读，读自己想读，思自己想思，纠正知识体系，对照并分辨，存疑并求同，分享并改变世界。我们不只完成了阅读，更在阅读中升华自己。
不在阅读中，寻求心理安慰，不将视线局限。
信息爆炸，阅读是那么方便，知其然和所以然，去发现，努力，改变世界真的不是随随便便就能做到，去积累，去find。
